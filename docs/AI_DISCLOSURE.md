# AI Assistance Disclosure

This project was developed through a heavily AI-assisted workflow directed by the author. Large language model tools were used extensively across:
- research planning and experiment framing
- code generation, revision, and refactoring support
- debugging assistance
- manuscript drafting and editing
- documentation cleanup
- LaTeX/PDF preparation

The author's role was to direct and manage the workflow, evaluate outputs, validate experiments and reported results, integrate the final materials, and make final release and manuscript decisions.

AI systems were used as tools and were not treated as authors. Responsibility for the final scientific content, interpretations, and release decisions rests with the author.
